# Online Banking System

## Project Overview
Online Banking System is a Java-based application that simulates basic banking operations. Users can register, login, create bank accounts, deposit money, withdraw money, transfer funds, and view transaction history.

## Features
- User Registration & Login
- Create Bank Account
- Check Balance
- Deposit Money
- Withdraw Money
- Transfer Money
- Transaction History
- MySQL Database Integration

## Technologies Used
- Java
- JDBC
- MySQL
- OOP Concepts

## Database Tables
- users
- accounts
- transactions

## Project Structure

DBConnection.java
User.java
Account.java
Transaction.java
AuthService.java
BankingService.java
Main.java
banking.sql

## How to Run
1. Create MySQL database using banking.sql
2. Configure database credentials in DBConnection.java
3. Add MySQL JDBC Driver
4. Compile all Java files
5. Run Main.java

## Concepts Used
- Classes & Objects
- Encapsulation
- JDBC Connectivity
- Exception Handling
- Database Management

## Future Enhancements
- Admin Panel
- Password Encryption
- GUI using Java Swing
- PDF Bank Statements
